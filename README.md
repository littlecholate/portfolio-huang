## How to Run

要安裝 nodejs & npm

1. `git clone`
2. `npm install`
3. `npm run dev`
4. 網頁預設會跑在 localhost:3000

改完後再 commit & push

## File Structure

-components/ 放 React Components (主要依頁面分類) <br />
-pages/ 放路由頁面 (Next.js 在 pages 下的檔案名稱即為網址) <br />
-public/ 放 public assets (圖片、影片) <br />
-styles/ 放 tailwind 客製化 css <br />
